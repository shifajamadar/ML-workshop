# 🚀 Indian Startup Funding Analysis

## 📌 Project Overview

This project performs an Exploratory Data Analysis (EDA) on the **Indian Startup Funding** dataset. The raw CSV dataset was cleaned, transformed, and analyzed to uncover valuable insights about startup funding trends in India.

The objective of this project is to convert raw startup funding data into meaningful information using Python and data analysis techniques.

---

## 📂 Dataset

- **Dataset:** Indian Startup Funding
- **File Format:** CSV
- **Records:** 5,074
- **Features:** 10

### Dataset Columns

- Startup Name
- Industry (Vertical)
- City
- State
- Investors
- Funding Round
- Funding Amount
- Funding Year
- Serial Number

---

## 🎯 Project Objectives

- Clean and preprocess raw startup funding data.
- Handle missing and inconsistent values.
- Perform exploratory data analysis (EDA).
- Identify startup funding trends.
- Analyze funding based on:
  - Year
  - Industry
  - City
  - State
  - Funding Round
  - Investors
- Generate meaningful insights from raw data.

## 📊 Data Preprocessing

The following preprocessing steps were performed:

- Imported the raw CSV dataset.
- Inspected dataset structure.
- Removed unnecessary columns.
- Checked missing values.
- Cleaned inconsistent records.
- Converted data into appropriate data types.
- Prepared the dataset for visualization and analysis.

---

## 📈 Exploratory Data Analysis

The project includes analysis of:

- Startup funding over the years
- Most funded startup sectors
- Top funded cities
- State-wise startup funding
- Funding round distribution
- Investor participation
- Funding amount distribution

Various visualizations were created to better understand the dataset.

---

## 📌 Key Insights

Some important insights obtained from the analysis include:

- Startup funding trends across different years.
- Cities attracting the highest investment.
- States with maximum startup activity.
- Most popular funding rounds.
- Industries receiving the highest funding.
- Frequently investing investors.
- Distribution of funding amounts among startups.
